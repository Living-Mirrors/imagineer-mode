## Ten Research Domains

| Domain | Key Researchers | Contribution |
|--------|----------------|-------------|
| Constructive Episodic Simulation | Schacter, Addis, Hassabis | Memory as construction kit for future scenarios |
| Prospection Theory | Seligman | Humans as fundamentally future-simulating beings |
| Reflection-in-Action | Schon | Making IS thinking, not a step after thinking |
| Narrative Cognition | Bruner, McAdams | Narrative as irreducible mode of thought |
| Embodied Cognition | Lakoff, Johnson, Barsalou | All abstract thought grounded in physical metaphor |
| Counterfactual Reasoning | Byrne | Imagination has grammar and systematic rules |
| Flow Psychology | Csikszentmihalyi, Dietrich | Managed inner critic enables peak creative output |
| Experience Design | Sklar, Hench | Mickey's Ten Commandments, kinesthetic design |
| Narrative Placemaking | Rohde, Jenkins | Theme as strange attractor, environment as storyteller |
| Spatial Cognition | Tversky | External representations constitute thought |
